1CDMS란? — 내용전문가가 하는 일

CDMS는 미림미디어랩의 콘텐츠 제작관리 시스템입니다. 인터넷만 되면 어디서든 담당 과목의 제작 상황과 영상을 볼 수 있습니다.

  • 내용전문가(교수자)님은 초대받은 과목만 보입니다. 다른 과목·다른 사업은 표시되지 않는 것이 정상입니다.
  • 주로 하시는 일은 ① 완성 영상(종편) 검수 — 영상을 보며 원하는 시점에 코멘트를 남기는 것, ② 원고·스토리보드·촬영교안·학습자료 확인입니다.
  • 남긴 검수 코멘트는 제작 담당자에게 자동으로 이메일 전달되므로 따로 정리해 보낼 필요가 없습니다.
  • 파일을 올리거나 지우는 기능은 보이지 않습니다 — 내용전문가 계정은 열람 전용입니다(자료가 실수로 바뀔 걱정이 없습니다).

2접속과 로그인 (처음부터 차근차근)

컴퓨터(권장)나 태블릿의 크롬(Chrome) 브라우저에서 사용합니다. 휴대폰도 되지만, 영상 위에 그림을 그리는 기능은 PC가 편합니다.

준비물

  • 계정(이메일)과 비밀번호 — 미림미디어랩 담당 PM이 이메일로 안내해 드립니다. 받지 못하셨다면 담당 PM에게 요청해 주세요.
  • 크롬 브라우저 — 엣지(Edge)도 사용 가능합니다.

접속하기

  1. 브라우저를 열고 주소창에 cdms.mirimmedialab.co.kr 를 입력한 뒤 Enter.
  2. 안내받은 이메일 주소와 비밀번호를 입력하고 로그인 을 누릅니다.
  3. 로그인이 되면 내가 초대된 과목 목록이 바로 보입니다.
즐겨찾기 등록 — 주소창 오른쪽의 ☆ (별표)를 누르면 다음부터 클릭 한 번으로 접속할 수 있습니다.
비밀번호를 잊으셨나요? 담당 PM에게 연락 주시면 재설정해 드립니다. 로그아웃은 왼쪽 메뉴 맨 아래 ⏏ 로그아웃 입니다.

3내 과목 열기

로그인하면 초대된 과목이 사업(개발 사업명) 아래에 표시됩니다.

  1. 목록에서 사업명을 누르면 그 안의 과목(과정)이 펼쳐집니다.
  2. 과목 이름을 클릭하면 그 과목의 진행표로 들어갑니다.
  3. 여러 과목에 초대되셨다면 모두 목록에 보입니다. 위쪽 과목 선택 드롭다운으로 빠르게 이동할 수도 있습니다.
  • 화면 왼쪽 위 미림미디어랩 CDMS 로고를 누르면 언제든 처음 화면으로 돌아옵니다.
  • 관리자용 통계 카드·품의 표시 등은 내용전문가 화면에는 나타나지 않습니다 — 정상입니다.

4진행표에서 검수할 차시 찾기

과목에 들어가면 차시(주차) × 제작 단계 표가 보입니다. 내용전문가에게는 원고 · 스토리보드 · 촬영교안 · 종편(완성 영상) · 학습자료 열만 표시됩니다.

  1. 표에서 검수할 차시 줄을 클릭합니다.
  2. 오른쪽에 열리는 상세 창에서 🎬 영상검수 버튼을 누릅니다 — 다음 장에서 자세히.
  3. 원고·스토리보드 파일을 보려면 각 단계 옆 📁 를 누릅니다 — 6장 참고.
종편 칸에 날짜가 있으면 완성 영상이 등록됐다는 뜻입니다 — 검수를 시작하실 수 있습니다. 날짜가 없으면 아직 제작 중입니다.

5🎬 영상 검수 — 핵심 기능

영상을 보면서 원하는 시점·위치에 그림과 코멘트를 남기는 화면입니다. 남긴 의견은 그대로 제작진에게 전달됩니다.

기본 사용 순서

  1. 차시 상세에서 🎬 영상검수 를 누르면 영상이 열립니다(클립이 여러 개면 이어서 재생됩니다).
  2. 영상을 보다가 지적할 지점에서 일시정지합니다(영상 클릭 또는 Space).
  3. 필요하면 왼쪽 도구로 화면 위에 표시합니다 — ✎ 를 켠 뒤 ╱ 선·▢ 박스·➤ 화살표·📍 핀 중 골라 마우스로 그립니다(색상 변경 가능).
  4. 오른쪽 아래 입력창에 의견을 씁니다. Enter = 줄바꿈이므로 여러 내용을 줄 바꿔 써도 됩니다.
  5. ＋ 남기기 버튼(또는 Ctrl+Enter)을 누르면 지금 시점(타임스탬프)에 코멘트가 저장됩니다.

알아두면 좋은 것

  • 타임라인의 노란 점(●)이 코멘트 지점입니다 — 점이나 코멘트의 시간을 클릭하면 그 장면으로 바로 이동합니다.
  • 그린 박스·화살표와 말풍선은 그 코멘트의 시간 구간에서만 화면에 나타납니다.
  • 코멘트를 남기는 즉시 제작 담당자에게 이메일로 전달됩니다(과목·차시·시점·내용 포함). 별도로 정리해 보내지 않으셔도 됩니다.
  • 제작진이 답글을 달면 코멘트 아래에 표시되고, 잘못 쓴 본인 코멘트는 수정·삭제할 수 있습니다.
  • 수정본이 다시 올라오면 같은 화면에서 새 버전으로 검수합니다. 버전 비교하기 를 누르면 이전본과 수정본을 나란히 재생해 비교할 수 있습니다.
검수 팁 — 한 지점에 지적할 내용이 여러 개면 줄을 바꿔 한 코멘트에 쓰는 것이 좋고, 다른 장면에 대한 내용이면 그 장면으로 이동해 따로 남겨주세요. 시점이 정확할수록 수정이 빨라집니다.

6원고·스토리보드·학습자료 확인(다운로드)

차시 상세에서 각 단계 옆 📁 를 누르면 그 차시의 파일을 볼 수 있습니다. 내용전문가는 원고 · 스토리보드 · 촬영교안 · 종편 · 학습자료 폴더를 열람할 수 있습니다.

  1. 차시 상세에서 보고 싶은 단계(예: 스토리보드)의 📁 를 누릅니다.
  2. 파일 오른쪽 ⬇ 를 누르면 등록된 원본 파일명 그대로 다운로드됩니다.
  3. 여러 파일이 필요하면 파일 앞 체크박스를 여러 개 선택(Shift+클릭 = 범위 선택)하고 ⬇ 선택 다운로드 — ZIP 파일 하나로 묶여 한 번에 받아집니다.
  4. PSD·이미지 파일은 👁 를 누르면 다운로드 없이 미리보기가 열리고, 🖼 썸네일 로 폴더 전체를 훑어볼 수 있습니다.
업로드·삭제 버튼이 안 보여요? 정상입니다. 내용전문가 계정은 열람 전용이라 자료를 바꾸거나 지울 수 없습니다.

7스토리보드 검수 의견 보내기

스토리보드(또는 원고) 검수는 파일을 내려받아 검토한 뒤 의견을 전달합니다. 전달 방법은 두 가지입니다.

방법 ① 이메일 회신 (가장 많이 쓰는 방법)

  1. 6장 방법으로 스토리보드 파일(PPT 등)을 다운로드합니다.
  2. 파일을 열어 검토하고, 수정 의견을 메모·표시합니다(PPT 메모 기능 또는 별도 문서).
  3. 검토한 파일(또는 의견 정리)을 담당 PM 이메일로 회신합니다 — 처음 계정을 안내드린 그 담당자입니다.

방법 ② CDMS에서 바로 남기기

  • 해당 차시의 영상이 이미 있다면, 🎬 영상검수 코멘트로 스토리보드 관련 의견을 남기셔도 담당자에게 전달됩니다.
  • 영상이 아직 없는 초기 단계에서는 이메일 회신(방법 ①)이 기본입니다.
어느 쪽이든 좋습니다 — 편하신 방법으로 보내주시면 제작진이 정리해 반영합니다. 급한 건은 담당 PM에게 전화 주셔도 됩니다.

8💬 불편 사항 접수

CDMS 사용 중 불편한 점이나 개선 아이디어는 화면 왼쪽 아래 💬 CDMS 불편 사항 접수 버튼으로 알려주세요.

  1. 버튼을 누르고 내용을 입력합니다.
  2. 필요하면 📷 화면 영역 캡처 로 문제 화면을 붙이거나 🖼 이미지 첨부 로 파일을 올립니다.
  3. 💾 의견 등록 을 누르면 접수 완료 — 처리 결과와 답변은 이메일로 알려드립니다.
  • 접수 목록에는 본인이 등록한 건만 표시됩니다.

9자주 묻는 질문(FAQ)

Q. 로그인이 안 돼요 / 비밀번호를 잊었어요.

이메일 주소의 오타를 먼저 확인해 주세요. 그래도 안 되면 담당 PM에게 연락해 주세요 — 비밀번호를 재설정해 드립니다.

Q. 로그인했는데 과목이 하나도 안 보여요.

아직 과목에 초대되지 않은 상태입니다. 담당 PM에게 초대를 요청해 주세요. 참고로 초대된 과목만 보이는 것이 정상입니다.

Q. 종편(영상) 칸이 비어 있어요 / 영상검수에 영상이 없다고 나와요.

아직 완성 영상이 등록되기 전입니다. 등록되면 진행표 종편 칸에 날짜가 표시됩니다. 일정이 궁금하시면 담당 PM에게 문의해 주세요.

Q. 코멘트를 잘못 남겼어요.

본인이 남긴 코멘트는 코멘트 옆 메뉴에서 수정·삭제할 수 있습니다.

Q. 내 코멘트가 잘 전달됐는지 어떻게 아나요?

등록되면 오른쪽 목록에 바로 나타나고, 동시에 제작 담당자에게 이메일이 발송됩니다. 담당자가 확인하면 답글이나 완료 표시가 달립니다.

Q. 파일 업로드 버튼이 안 보여요.

내용전문가 계정은 열람 전용입니다. 전달할 파일이 있으면 담당 PM에게 이메일로 보내주세요.

Q. 휴대폰으로도 되나요?

영상 재생·코멘트 열람은 가능하지만, 그리기 도구와 파일 검토는 PC 크롬을 권장합니다.
