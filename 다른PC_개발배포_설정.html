CDMS 개발·배포 — 다른 PC에서 하기

저장소를 받아 수정하고, 저장하면 자동 배포됩니다

0먼저 이해하기 — 배포는 이미 자동입니다

코드를 GitHub에 올리기(push) 만 하면, 화면과 서버 기능이 자동으로 배포됩니다. 배포에 필요한 열쇠(토큰)는 PC가 아니라 GitHub에 저장되어 있어서, 다른 PC엔 “저장소를 받아 올리는 환경”만 있으면 됩니다.

대상어디에 있나배포 방법
화면(프런트)저장소의 cdms-deploy/push → Vercel 자동배포
서버 기능(Edge Functions)supabase/functions/push → GitHub Actions 자동배포
DB 변경(마이그레이션)supabase/migrations/GitHub Actions에서 수동 실행
저장소 주소: github.com/ghnam7312-droid/cdms (비공개). 이 계정으로 로그인해야 접근됩니다.

A권장 방법 — GitHub Desktop (가장 쉬움)

명령어 없이 화면 클릭으로 저장소를 받고, 수정 후 “Commit → Push”만 하면 됩니다.

  1. GitHub Desktop 설치 — desktop.github.com 에서 다운로드 후 설치.
  2. 실행 후 Sign in to GitHub.com 으로 ghnam7312-droid 계정(또는 저장소 접근 권한이 있는 계정)으로 로그인.
  3. File → Clone repository → 목록에서 cdms 선택 → 저장 위치 지정 → Clone.
  4. 파일 수정(메모장/VS Code 등). 저장하면 GitHub Desktop 왼쪽에 바뀐 파일이 표시됩니다.
  5. 아래 Summary 칸에 변경 내용을 쓰고 Commit to main 클릭.
  6. 오른쪽 위 Push origin 클릭 → 1~2분 뒤 자동배포 완료.
코드 편집기는 VS Code(code.visualstudio.com) 를 함께 설치하면 편합니다. GitHub Desktop의 Open in Visual Studio Code 로 바로 열 수 있습니다.

B대안 — 설치 없이 웹에서 바로 수정

간단한 수정은 브라우저만으로 가능합니다(설치 불필요).

  1. 브라우저에서 github.com/ghnam7312-droid/cdms 접속 → 로그인.
  2. 수정할 파일을 열고 오른쪽 위 연필(✏) Edit 클릭.
  3. 내용 수정 후 아래 Commit changes → 자동배포.
웹 편집은 한 파일씩만 편하고, 실수 시 되돌리기가 번거롭습니다. 여러 파일·본격 작업은 방법 A를 권장합니다.

C명령줄(Git) 방법 + 원클릭 배포

터미널이 익숙하면 Git으로 클론 후, 첨부한 배포.bat 을 더블클릭해 올릴 수 있습니다.

  1. Git 설치 — git-scm.com.
  2. 폴더에서 클론:
git clone https://github.com/ghnam7312-droid/cdms.git
cd cdms

인증 창이 뜨면 ghnam7312-droid 계정으로 로그인(또는 토큰 입력)합니다.

  1. 수정 후, 저장소 폴더의 배포.bat 을 더블클릭 → 커밋 메시지 입력 → 자동으로 add·commit·push.

또는 명령으로 직접:

git add -A
git commit -m "수정 내용"
git push origin main

D배포 확인·마이그레이션·주의

  • 배포 진행 확인 — 저장소의 Actions 탭(서버 기능)과 Vercel 대시보드(화면)에서 상태를 볼 수 있습니다. 보통 1~2분.
  • DB 마이그레이션(구조 변경) — Actions → Deploy Supabase → Run workflow 에서 run_migrations 체크 후 실행(필요할 때만).
  • 화면이 안 바뀌면 — 브라우저 강력 새로고침 Ctrl+F5.
보안 — 저장소는 비공개입니다. 접근 토큰·비밀번호는 공유 PC나 메신저에 남기지 마세요. 다른 사람이 쓸 PC라면 작업 후 GitHub Desktop/브라우저에서 로그아웃 하세요.
여러 PC에서 작업할 땐, 시작 전 항상 Fetch/Pull(GitHub Desktop) 또는 git pull 로 최신본을 먼저 받으세요. 충돌을 예방합니다.
